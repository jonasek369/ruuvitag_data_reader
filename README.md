# Why?

I couldn’t get the `ruuvitag-sensor` Python library to work (probably a problem on my end), so I recreated the basic functionality for receiving the data myself.

Parts of the code were adapted and repurposed from the official repository: [ruuvitag-sensor](https://github.com/ttu/ruuvitag-sensor).  
If you need a more complete and maintained solution, you should probably use that instead.
